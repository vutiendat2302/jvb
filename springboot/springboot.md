# Spring Boot 

- Sử dụng chính spring mvc bên trong 
- công cụ thiết lập, cấu hình và chạy một ứng dụng Spring một cách tự động 
- Cấu hình sẵn khi thêm dependence 
- là một công cụ đóng gói và tự động cấu hình không giống với Spring mvc là một web framework 

1. Spring Boot 
- Yêu cầu hệ thống: 4.1.0 từ java 17 trở lên và tương thích với java 26. 
- Công cụ xây dựng: hỗ trợ maven 3.6.3 trở lên hoặc gradle 8.x 
- Bộ chứa servlet: hỗ trợ các bộ chứa: Tomcat 11.0.x, hoặc Jetty 12.1.x phiên bản servlet 6.1 

- Spring Boot CLI là giao diện dòng lệnh (Comand Line Interface) dùng để  chạy nhanh một dự file groovy, chủ yếu test tính năng. Dự án thực tế không dùng CLI. Dùng java, quản lý bằng maven/gradle
